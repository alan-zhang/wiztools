Last Request Snoop App
======================

Note: This is a Servlet 3.0 web-application. Please use a
compatible server.

What?
-----

This application displays the last request made to the URL:

http://host:port/snoop/snoop/

Why?
----

Use it for verifying what request was posted last to a URL.

Build
-----

Uses Maven for building. To build:

$ mvn package

This creates the war: snoop.war.
